### 1.2 桌面环境清理

执行以下设置（拍摄前 10 分钟做）：

1. **关闭所有通知**
   - macOS：System Settings → Notifications → 勿扰模式 ON
   - Windows：设置 → 系统 → 通知 → 专注助手 ON
2. **关闭社交软件**：微信 / QQ / 钉钉 / 邮箱客户端全退出
3. **桌面壁纸**：换成纯色（推荐黑色 #1a1a1a 或纯白），不要个人照片
4. **隐藏 Dock / 任务栏**
   - macOS：System Settings → Desktop & Dock → Automatically hide
   - Windows：任务栏 → 任务栏行为 → 自动隐藏
5. **屏幕分辨率**：1920×1080（不要 4K，文件过大）
6. **关闭浏览器扩展**：使用 Chrome 隐身模式或新 profile，避免扩展图标干扰

### 3.3 避免敏感数据

- `old_user` 账号有 60 道你的真实错题，**注意 AI 分析里不要暴露任何个人信息**
- 如果某次 AI 输出包含真实姓名 / 学号，**剪辑时打码或重新生成**
- 浏览器 URL 不要泄露内网 IP（用 `127.0.0.1` 是安全的）

### 3.4 节奏控制

- 每段旁白与画面动作同步
- 建议**先录无声画面 → 后期配音**（更精准，不会卡壳）
- 总时长 5 分钟整，**不要超过 5 分 30 秒**

### 3.5 容错策略

- 录的时候**多录几条**（每个 Scene 录 2-3 次取最佳）
- 如果某个画面 NG（比如 AI 接口卡了 / 输错字了），不要停录，**重新做这个动作**，事后剪掉
- 每个 Scene 录之前**先看一遍分镜稿**

## 4. 剪辑与导出建议

### 4.1 剪辑步骤

1. **导入素材**：把所有录屏导入 DaVinci Resolve（或剪映）
2. **粗剪**：按 Scene 顺序排列，剪掉 NG 镜头
3. **配音**（如果先录画面后配音）：用麦克风录制旁白，分段录制 6 个 Scene
4. **同步**：把旁白对到画面动作
5. **加 BGM**：低音量背景音乐（推荐 Lofi 或 Ambient 风格，**无歌词**），音量 -25dB 以下
6. **加字幕**：每段旁白底部加白色字幕（黑色描边），字号大方便手机看
7. **加片头片尾**：3-5 秒静帧
8. **导出**

### 4.2 BGM 来源

免费可商用：

- [Pixabay Music](https://pixabay.com/music/) — 免费 Lofi / Ambient
- [YouTube Audio Library](https://studio.youtube.com/) — 免费 BGM 库
- 注意：所选 BGM 要在视频结尾或描述里**标注音乐来源**（详见 ORIGINALITY.md §4.4）

### 4.3 导出参数

```
Format: MP4
Resolution: 1920x1080
FPS: 30
Codec: H.264
Bitrate: 6-8 Mbps
Audio: AAC 192 kbps
File size: 150-300 MB
```

## 7. 常见问题 FAQ

**Q1：录的视频文件太大怎么办？**
A：用 HandBrake（免费）压缩。Preset 选 "Fast 1080p30"，码率调到 4-6 Mbps。

**Q2：AI 分析输出太慢，画面停太久怎么办？**
A：剪辑时**加速 1.5-2 倍**（用 DaVinci 的 Speed Change 功能），并加一个"AI 正在分析中..."的文字提示。

**Q3：录制时麦克风有杂音怎么办？**
A：用 Audacity（免费）做降噪：录一段安静背景 → Effects → Noise Reduction → Get Noise Profile → 应用到全段。

**Q4：旁白普通话不标准怎么办？**
A：用 AI 配音工具（如剪映自带的"文本朗读"功能），把分镜稿的旁白文字粘贴进去，选标准男声 / 女声。

**Q5：分镜里某个画面录不出来怎么办？**
A：跳过那段，用文字字幕代替说明，比强行录一个 NG 的画面更专业。
