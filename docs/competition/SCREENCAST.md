### 1.2 桌面环境清理

执行以下设置（拍摄前 10 分钟做）：

1. **关闭所有通知**
   - macOS：System Settings → Notifications → 勿扰模式 ON
   - Windows：设置 → 系统 → 通知 → 专注助手 ON
2. **关闭社交软件**：微信 / QQ / 钉钉 / 邮箱客户端全退出
3. **桌面壁纸**：换成纯色（推荐黑色 #1a1a1a 或纯白），不要个人照片
4. **隐藏 Dock / 任务栏**
   - macOS：System Settings → Desktop & Dock → Automatically hide
   - Windows：任务栏 → 任务栏行为 → 自动隐藏
5. **屏幕分辨率**：1920×1080（不要 4K，文件过大）
6. **关闭浏览器扩展**：使用 Chrome 隐身模式或新 profile，避免扩展图标干扰

### 1.3 macOS 自带屏幕录制设置

#### 调出录屏工具

按 **`⌘ Shift 5`**，屏幕底部会弹出工具栏：

```
[📷 全屏截图] [📷 区域截图] [🎬 全屏录屏] [🎬 区域录屏] [选项▼] [录制]
                                ↑                            ↑           ↑
                            首选这个              关键设置在这里      开拍
```

#### 关键设置（点 "选项 ▼"）

| 选项 | 推荐值 | 说明 |
|---|---|---|
| 保存到 | **桌面** | 方便找文件 |
| 计时器 | **无** | 不需要倒计时 |
| 麦克风 | **选择内置麦克风（如 MacBook 麦克风）** | ⚠️ 默认是"无"，必须手动选择，否则录出来是无声片！ |
| 在录制中显示鼠标点击 | ✅ **打开** | 评委能看到你点了哪里，更直观 |
| 在录制中显示触控板 | 关 | — |
| 记住上次的选择 | ✅ 打开 | 多次录制保持配置 |

#### 录制流程

1. **选择"区域录屏"**（第 4 个图标），不要选"全屏"——区域录屏可以避开 Dock / 菜单栏，剪辑省事
2. 拖出录制框，**只盖住浏览器窗口**，不带状态栏 / Dock
3. 点 **录制** 按钮 → 开拍
4. 录完按 **`⌘ Control Esc`**（或点菜单栏右上角的 ⏹️ 图标）停止
5. 视频自动存到桌面，文件名形如 `屏幕录制 2026-05-17 上午10.30.mov`

#### 几个 macOS 自带录屏的"坑"与解决

| 问题 | 解决 |
|---|---|
| **录到的视频是 `.mov`，不是 `.mp4`** | 大多数视频平台和评审系统都吃 `.mov`；如果非要 `.mp4`，剪映导出时直接选 `mp4` 即可，不需要额外转换 |
| **录不到系统声音**（macOS 隐私限制） | 本项目演示视频不需要系统声音；旁白用麦克风录，或用剪映 AI 朗读代替。**如必须录系统声音**：免费装 [BlackHole](https://github.com/ExistentialAudio/BlackHole)（虚拟音频路由），约 5 分钟配置 |
| **Retina 屏录出来分辨率 2 倍，文件超大（5min 录屏可能 1-2 GB）** | 区域录屏框小一些（800×600 即可）；或录完用 [HandBrake](https://handbrake.fr/)（免费）压缩到 200-300 MB |
| **录制中按了 Mission Control / 切桌面，画面会跳** | 拍摄前关闭所有不相关的窗口；录制时只在录屏框内操作 |
| **想看自己鼠标轨迹但忘开"显示鼠标点击"** | 录制前一定要进 "选项" 确认勾选 |

#### 推荐的拍摄方式："先画面后配音"

新手最容易卡的是**边操作边说话**——既要看屏幕又要想词，画面节奏会乱。推荐：

1. **第一遍**：纯录画面，不说话，按分镜稿做完每个 Scene 的操作
2. **第二遍**：在剪映里把分镜稿的旁白文字粘进 "文本"，点 "文本朗读"（剪映自带 AI 朗读），选标准男声或女声，自动生成配音
3. **第三遍**：把 AI 配音对到画面动作上，加字幕（剪映"识别字幕"功能自动出字幕）

这个流程**无需开口录音**，全程可以一个人在宿舍里完成，质量比真人念稿更稳定。

### 3.2 文字停留时间

| 内容类型 | 停留 |
|---|---|
| 错题列表 / Dashboard 等"展示型"画面 | 2-3 秒 |
| 代码 / 题面 | 1-2 秒 |
| 弹窗 / 二次确认 | 1 秒 |
| 标题静帧（片头 / 片尾） | 3-5 秒 |

### 3.3 避免敏感数据

- `old_user` 账号有 60 道你的真实错题，**注意 AI 分析里不要暴露任何个人信息**
- 如果某次 AI 输出包含真实姓名 / 学号，**剪辑时打码或重新生成**
- 浏览器 URL 不要泄露内网 IP（用 `127.0.0.1` 是安全的）

### 3.4 节奏控制

- 每段旁白与画面动作同步
- 建议**先录无声画面 → 后期配音**（更精准，不会卡壳）
- 总时长 5 分钟整，**不要超过 5 分 30 秒**

### 3.5 容错策略

- 录的时候**多录几条**（每个 Scene 录 2-3 次取最佳）
- 如果某个画面 NG（比如 AI 接口卡了 / 输错字了），不要停录，**重新做这个动作**，事后剪掉
- 每个 Scene 录之前**先看一遍分镜稿**

## 4. 剪辑与导出建议

### 4.1 剪辑步骤

1. **导入素材**：把所有 `.mov` 录屏文件拖进剪映 Mac 版（或 DaVinci Resolve）
2. **粗剪**：按 Scene 顺序排列，剪掉 NG 镜头和录制工具栏出现的瞬间
3. **配音**（推荐 AI 朗读，零开麦门槛）：
   - 在剪映里新建"文本"图层，把分镜稿的旁白文字逐段粘进去
   - 点 "文本朗读"，选标准男声或女声（推荐"东北老铁" / "解说小帅" / "标准男声"）
   - 自动生成的配音与文字时长精确对应
   - 或者：用麦克风分段录 6 个 Scene 的真人旁白
4. **同步**：把 AI 配音（或真人旁白）拖到对应 Scene 画面下方，调整时间轴对齐
5. **加字幕**：剪映 → "识别字幕" → 自动从音轨提取，1 秒出全片字幕；统一改成白色字号 36，黑色描边
6. **加 BGM**：低音量背景音乐（推荐 Lofi 或 Ambient 风格，**无歌词**），音量调到 -25dB 以下
7. **加片头片尾**：3-5 秒静帧字幕（项目名 + 团队 + 仓库链接）
8. **导出**：见 §4.3

### 4.2 BGM 来源

免费可商用：

## 7. 常见问题 FAQ

**Q1：录的视频文件太大怎么办？**
A：用 HandBrake（免费）压缩。Preset 选 "Fast 1080p30"，码率调到 4-6 Mbps。

**Q2：AI 分析输出太慢，画面停太久怎么办？**
A：剪辑时**加速 1.5-2 倍**（用 DaVinci 的 Speed Change 功能），并加一个"AI 正在分析中..."的文字提示。

**Q3：录制时麦克风有杂音怎么办？**
A：用 Audacity（免费）做降噪：录一段安静背景 → Effects → Noise Reduction → Get Noise Profile → 应用到全段。

**Q4：旁白普通话不标准怎么办？**
A：用 AI 配音工具（如剪映自带的"文本朗读"功能），把分镜稿的旁白文字粘贴进去，选标准男声 / 女声。

**Q5：分镜里某个画面录不出来怎么办？**
A：跳过那段，用文字字幕代替说明，比强行录一个 NG 的画面更专业。

**Q6：macOS 自带录屏录出来声音很小 / 没声音？**
A：99% 是 `⌘ Shift 5` → "选项 ▼" → "麦克风" 没选。默认是"无"，必须手动选择"MacBook 麦克风"或外接麦才会录上声音。**先录 10 秒试拍一段播放确认有声音再正式开拍**。

**Q7：剪映导出后视频时长超过 5 分钟怎么办？**
A：剪映右下角时间轴看总时长。常见超时原因：Scene 3 的 AI 流式输出太慢（占 30+ 秒）。解决方案：选中那段 AI 输出片段 → "变速" → 1.5x 或 2x，并在画面上贴一个"AI 实时分析中..."的文字提示。

**Q8：录制时不小心录到了通知 / 弹窗 / QQ 消息怎么办？**
A：剪辑时直接剪掉那几秒，或用剪映的"画中画"功能盖住那块区域。**录制前再次确认勿扰模式已开**。
