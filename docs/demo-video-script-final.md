# CodeRecall (码迹) - 演示视频脚本 v2

**总时长**: ~5分钟
**基调**: 专业、流畅，展示核心解决问题的能力（Golden Path）。

## Scene 1 (0:00-0:30): 应用介绍与仪表盘概览
- **时间戳**: 0:00 - 0:30
- **屏幕状态**: 首页 Dashboard，展示整体统计数据、今日待复习数量、学习热力图。
- **鼠标动作**: 鼠标平滑地在统计卡片和热力图上悬停，展示 Tooltip。
- **旁白 (Narration)**:
  “大家好，欢迎了解 CodeRecall 码迹——一款专为程序员打造的错题本与知识管理工具。对于开发者来说，踩过的坑如果不好好记录，很容易反复掉进同一个陷阱。在我们的仪表盘上，你可以直观地看到当前的掌握情况、热力图打卡记录，以及今天需要复习的代码错题。”

## Scene 2 (0:30-1:30): 添加新错题 (React Hooks 闭包陷阱)
- **时间戳**: 0:30 - 1:30
- **屏幕状态**: 点击“添加错题”，进入错题录入表单。
- **鼠标动作**: 输入标题、选择标签(React)、在 Monaco Editor 中分别粘贴错误代码和正确代码，填写反思，点击保存。
- **旁白 (Narration)**:
  “让我们来记录一个经典的错误。比如这个 React Hooks 的闭包陷阱问题。我们在编辑器中分别贴入修改前导致 Bug 的代码，和修复后的正确代码。CodeRecall 内置了 Monaco 编辑器和 Diff 视图，能够原汁原味地保留你的代码格式。写下我们的反思和标签后，保存这条记录，它就进入了我们的知识库。”

## Scene 3 (1:30-2:30): 间隔重复复习流
- **时间戳**: 1:30 - 2:30
- **屏幕状态**: 从仪表盘点击“开始复习”，进入专注复习模式。
- **鼠标动作**: 屏幕只显示错误代码和现象描述。停顿几秒后，点击“显示答案”。查看对比后，点击“熟练(4分)”按钮。
- **旁白 (Narration)**:
  “CodeRecall 的核心是基于 SM-2 算法的间隔重复机制。系统会根据遗忘曲线自动安排复习。进入复习模式后，系统先只展示错误代码。这时候我们在脑海中回想原因。点击‘显示答案’，就可以看到 Diff 对比和解析。根据刚才的回想情况，我给自己打 4 分。系统会据此动态计算下一次这道题出现的时间。”

## Scene 4 (2:30-3:30): AI 深度分析功能
- **时间戳**: 2:30 - 3:30
- **屏幕状态**: 在某条错题详情页，点击“AI 深度分析”按钮。
- **鼠标动作**: 点击按钮，等待 SSE 流式输出开始，鼠标跟随向下滚动的文本滑动。
- **旁白 (Narration)**:
  “有时候，光有自己写的反思还不够。我们接入了 AI 分析功能。点击这里，AI 会结合你提供的错误和正确代码，流式输出深度解析。它不仅会解释底层原理，还会推荐最佳实践和相关的系统性知识点。注意看，输出过程非常流畅，页面也会自动跟随滚动到底部，体验就像有一位资深工程师在旁边手把手指导。”

## Scene 5 (3:30-4:30): 统计面板与弱点分析
- **时间戳**: 3:30 - 4:30
- **屏幕状态**: 切换到左侧导航的“统计 (Stats)”页面。
- **鼠标动作**: 滚动查看不同图表，重点指向“知识薄弱区域 (Weak areas)”雷达图或条形图。
- **旁白 (Narration)**:
  “随着使用时间的增加，你积累的数据会形成极具价值的个人能力图谱。在统计面板中，图表实时反映了你的复习趋势和掌握度分布。尤其这个‘薄弱区域’分析，它可以精准指出你在哪些技术栈（比如 Python 算法或者并发编程）上犯错最多，帮助你有针对性地进行专项突破。”

## Scene 6 (4:30-5:00): 导入导出与总结
- **时间戳**: 4:30 - 5:00
- **屏幕状态**: 进入设置页，演示一键导出 JSON。
- **鼠标动作**: 点击“导出数据”，浏览器下载文件。随后回到首页。
- **旁白 (Narration)**:
  “最后，你的数据完全属于你自己。CodeRecall 提供完整的 JSON 导入导出功能，方便你进行数据备份或迁移。总结来说，通过结构化记录、智能复习算法和 AI 辅助，CodeRecall 帮助你把 Bug 转化为真正的经验。感谢观看！”
